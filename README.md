## üíª Local Setup and Installation

To review the code and run the project locally, follow these steps:

### Prerequisites

- Node.js (version 18 or higher)

- Package manager (pnpm, npm, or yarn)

### Steps

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/mohammademami2005/portfolio.git
   cd portfolio
   ```

1. **Install Dependencies:**

   ```bash
   npm install
   # or pnpm install
   ```

1. **Run the Project:**

   ```bash
   npm run dev
   # or pnpm dev
   ```

   The project will be accessible at `http://localhost:3000`.
